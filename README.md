
# Gramadóir Andróidigh

An app to help you write better messages and short texts as Gaeilge. 

It follows a workflow I use on desktops where I write the text as Gaeilge, check it in something like Google Translate to make sure I'm not way off the mark and then revise if necessary. Then I use [*An Gramadóir*](https://cadhan.com/gramadoir/index-en.html) to check that the text actually makes sense, and to correct grammatical errors.

I made this to scratch my own itch, and to allow me to use a workflow on my phone that I was following on the desktop. 
## FAQ

#### Why no English to Irish translation function?

Online English to Irish tranlsation is janky and innaccurate. You'll produce worse results than if you start in Irish, and you won't learn much either.

#### What if I don't know the Irish for a word I want to use?

Check out the exellent online dictionaries at https://focloir.ie and https://teanglann.ie
